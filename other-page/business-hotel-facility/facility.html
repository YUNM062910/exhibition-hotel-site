Hotel Facilities

快適な滞在を支える館内施設

ビジネスでのご利用やご家族でのご宿泊にも安心してお過ごしいただけるよう、 便利で使いやすい館内設備をご用意しております。

コインランドリー

  • 営業時間：6:00〜24:00
  • 場所：2階 コインランドリーコーナー
  • 洗濯機：6台 / 1回:200円（硬貨のみ対応）
  • 乾燥機：6台 / 10分:200円（硬貨のみ対応）
  • 洗剤：自動投入式のため不要

ご利用上の注意

・混雑時は譲り合ってご利用ください。

・洗濯物の長時間放置はご遠慮ください。

・ホテル利用者のみのご利用とさせていただきます。

24時間案内可能のロビー

  • 場所：ホテル1階 フロント横
  • 対応時間：24時間
  • チェックイン・チェックアウトのご案内
  • 周辺施設や交通案内のご相談
  • 緊急時のご対応

ご利用上の注意

・時間により担当が席を外している可能性がございます。
（その際はフロントにありますコールボタンにてお知らせ下さい）

・アメニティは担当がいない際でも客室への持ち運びは可能です。

自動販売機コーナー

  • 設置種類：飲料 / 軽食 / アイス など
  • 場所：1,2階エレベーター横
  • 営業時間：24時間
  • 新硬貨・新紙幣：対応

ご利用時の注意

・商品詰まりやお支払いトラブルの際は、フロントまでお知らせください。

ワーキングスペース

  • 場所：5階 ラウンジスペース
  • 机の数：24席
  • 利用時間：6:00〜22:00
  • 貸出品：延長コード / 充電器 / デスクライト など

ご利用上の注意

・長時間の席取りや大きな音の出る作業はご遠慮ください。

・ホテル利用者のみのご利用とさせていただきます。

無料Wi-Fi

  • 館内全エリアで無料Wi-Fiをご利用いただけます
  • SSID：〇〇〇〇〇〇
  • パスワード：客室内案内をご確認ください
  • 接続方法：スマートフォン・PCから接続可能

ご利用時の注意

・通信速度は利用状況により変動します。
（セキュリティが必要な作業は各自で対策をお願いいたします）

・ホテル内でのみでのご使用をお願いいたします。

アメニティバー

  • ロビー設置：
    歯ブラシ / カミソリ / 綿棒 / ヘアブラシ / 館内衣服 など
  • 客室内設置：
    タオル / シャンプー / ボディソープ / ドライヤー / ヘアアイロン など
  • 必要な分だけお取りいただけます

ご利用時の注意

・追加が必要な場合はフロントへお声がけください。
